SHA Benchmark - STM32MP25

The benchmarks compare five implementations for SHA1, SHA256 and SHA512 :

  • af_alg and cryptodev Provider with/without optim
    • Pv-AF_ALG / Pv-Cryptodev (without optim) : baseline Provider
    • Pv-AF_ALG / Pv-Cryptodev (with optim) : optimized Provider
  • OpenSSL software
  • The legacy engine devcrypto

Each measured with across block sizes and over at least 5 seconds.
